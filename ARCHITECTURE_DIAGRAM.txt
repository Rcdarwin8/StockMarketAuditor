───────────────────

CALCULATION
───────────

VALIDATION
──────────
|calculated - website| = |5.31 - 5.31| = 0.00 ≤ 0.1 → ✓ MATCH


FILTERING
─────────
5.31% > 5.0% → HIGH PERFORMER → Include in Excel report

┌───────────────────────────────────────────────────────────────────────────┐
│  XPATH STRATEGY BREAKDOWN                                                 │
└───────────────────────────────────────────────────────────────────────────┘

Base Table XPath:
//table[contains(@class, 'mctable')]

Row Navigation (skip header at position 1):
//table[contains(@class, 'mctable')]//tr[2]     → First data row
//table[contains(@class, 'mctable')]//tr[3]     → Second data row
//table[contains(@class, 'mctable')]//tr[i+1]   → i-th data row (loop)

Cell Extraction (column indexing):
rowXPath + "/td[1]//a"   → Company Name (inside anchor tag)
rowXPath + "/td[3]"      → Previous Close Price
rowXPath + "/td[4]"      → Current Price
rowXPath + "/td[5]"      → Website's % Change


┌───────────────────────────────────────────────────────────────────────────┐
│  ERROR HANDLING STRATEGY                                                  │
└───────────────────────────────────────────────────────────────────────────┘

1. WebDriver Initialization
   └─→ Catch: WebDriverException
       └─→ Action: Log error, attempt fallback driver

2. Page Loading
   └─→ Catch: TimeoutException
       └─→ Action: Retry with increased wait time

3. Element Not Found
   └─→ Catch: NoSuchElementException
       └─→ Action: Skip row, log warning, continue

4. Data Parsing
   └─→ Catch: NumberFormatException
       └─→ Action: Return 0.0, log invalid data

5. Excel Generation
   └─→ Catch: IOException
       └─→ Action: Check permissions, log error

6. Empty Results
   └─→ Validation: if (stockList.isEmpty())
       └─→ Action: Display message, suggest threshold adjustment
